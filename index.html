01 About

I'm a software developer and Odoo consultant based in Baton Rouge, Louisiana. Most developers start by writing code — I start by finding the real problem. Workflow, data, or software: I figure out which one is costing you before I build anything.

My day-to-day work at Systems Services Inc. spans the full Odoo stack — sales, inventory, POS, accounting, and manufacturing — across versions 17, 18, and 19. From custom modules and version migrations to API integrations, I've worked across nearly every part of the platform, understanding not just the code but the business logic behind it.

I hold a BA in Computer Science, an MBA in Data Analytics, and a professional certificate in Data Science from MIT. That mix of engineering and business judgment is what I bring to every project — and it's the difference between software that works and software that actually solves the problem.
